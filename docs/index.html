Examples

These are examples of SIMProv working with dc.js visualizations. The collaboration examples have two versions (C1 and C2) so that they can be demoed in two tabs of the same browser (otherwise, the provenance stored in IndexedDB could overwrite itself). The streaming examples require the use of the streaming engine.
